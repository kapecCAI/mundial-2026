# Mundial 2026 — Comparativa de predicciones (Prode)

Sitio estático que compara el fixture completo del Mundial 2026 según **6 opiniones**:
**Claude · Codex · Gemini · Kimi · Maldini (YouTube) · Consenso**.

Cada pestaña tiene dos subpestañas:
- **Fixture**: 12 grupos (posiciones + partidos con resultado) y todo el cuadro de eliminatorias (16avos → final), con banderas.
- **Explicación**: el razonamiento de cada opinión.

> Predicciones generadas con datos a junio 2026. No son garantía: el fútbol tiene varianza.

## Deploy

Es HTML estático puro. En Vercel: *New Project → Import* este repo y **Deploy** (no requiere build; `index.html` se sirve en la raíz).

Las banderas se cargan desde `flagcdn.com` (requiere conexión).
